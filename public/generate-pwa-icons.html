PWA Icon Generator

Instructions

This utility helps you generate PNG icons from SVG files for your Progressive Web App.

  1. Click the "Generate Icons" button below to render the SVG icons to canvas elements.
  2. Right-click on each canvas and select "Save image as..." to save the PNG files.
  3. Save them with the names "pwa-192x192.png" and "pwa-512x512.png" in the public folder.
